# Wishlist-website

This repository is a static site intended for GitHub Pages.

## GitHub Pages entry file

GitHub Pages serves `index.html` by default. This repo now includes `index.html`, which redirects to `shopping-list-1.html`.

If deployment still does not show the site, verify repository settings:

1. Go to **Settings → Pages**.
2. Set **Build and deployment** source to **Deploy from a branch**.
3. Select branch **main** (or your default branch) and folder **/(root)**.
4. Save and wait for the Pages deployment to complete.
